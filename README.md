
# アプリモニター

## 内容

 - 超軽量のアプリ監視ツールです。

## 監視内容

 - 起動されたウィンドウを記録(6秒間隔)
 - 画面のロック、ロック解除を記録(随時)
 - モニター自体の稼働状況を記録(1分間隔)
 - PCシャットダウンを検知し、次回起動時に記録。
 - PCが異常停止した場合も、次回起動時に記録しますが時刻は次回起動時点のもの。

## 使い方

 - スタートアップにexeを登録してください。
 - タスクトレイのアイコンをクリックすると履歴画面が表示されます。
 - 画面更新にはウィンドウを閉じて開きなおしてください。
 - このアプリモニターには停止操作はありません。強引にkillするしかありません。
 - 強引にkillされたかは、PCシャットダウン時間とアプリモニターの最終稼働時間との比較で分かります。
